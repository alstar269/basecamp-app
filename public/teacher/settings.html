🔑 (선택) 교회 전용 키 등록하기 독립 할당량이 필요할 때만 ▾

대부분의 경우 운영자 공용 키로 충분합니다. 다음 경우에만 전용 키를 등록하세요:

  • 수련회 규모가 커서 공용 할당량과 공유하기 어려운 경우
  • 교회 고유의 데이터 처리 정책이 필요한 경우
  • 다른 AI 제공자(Claude 등)를 쓰고 싶은 경우

🆓 Groq 키 발급 (3분, 무료)

  1. console.groq.com/keys 접속 · Google 계정 로그인
  2. "Create API Key" 버튼 클릭
  3. 생성된 gsk_... 키를 아래에 붙여넣고 저장

🔑 API 키 등록

키는 AES-256 암호화 후 저장됩니다. 원본은 DB에서 복구할 수 없어요.

❓ 왜 교사가 직접 키를 등록하나요?

베이스캠프는 완전 무료로 제공되기 때문에, AI 비용은 각 교회가 직접 부담합니다. Google Gemini는 신용카드 없이도 무료 할당량만으로 수련회에 충분히 쓸 수 있어요. 각 교회가 독립된 키를 가지므로 다른 교회와 사용량을 나눠 쓰지 않습니다.
