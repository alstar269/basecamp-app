# 🎤 컨퍼런스 발표 슬라이드 초안 (15장)

> **주제**: 내 마음의 베이스캠프 — 청소년 수련회용 AI 성경 멘토 코칭 앱
> **발표 시간**: 25-30분 (Q&A 포함 시 35분)
> **대상**: 교회 교역자·청소년 담당 교사
> **URL**: https://basecamp-app-woad.vercel.app/

---

## 🎨 공통 디자인 가이드

- **컬러**: 보라(#8B7AD9) · 그린(#7EC4A5) 그라데이션 (앱과 동일)
- **폰트**: Pretendard (한글) / 굵은 타이틀 + 가벼운 본문
- **배경**: 흰색 or 연보라(#F4F0FC), 포인트 슬라이드만 그라데이션
- **이모지**: 슬라이드당 1-2개 (과하지 않게)
- **글꼴 크기**: 제목 44pt / 부제 28pt / 본문 20pt / 캡션 14pt

> "(청중 반응 기다린 뒤) 많아야 10분, 대부분은 0분입니다. 정작 수련회는 그 10분을 위해 존재하는데요."

   교사도 원문을 못 봅니다

   학생 대화 원문
          ↓
   [AI 카테고리 자동 분류]
          ↓
   [K≥5 익명성 필터]
          ↓
   교사 대시보드 (집계만)
   ├─ 고민 영역 분포
   ├─ 키워드 구름

```

## 발표자 노트 (2분)
> "많은 분들이 '교사가 학생 대화 다 볼 수 있으면 부담스럽지 않나?' 걱정하시는데, 베이스캠프는 **교사도 원문을 볼 수 없게** 설계됐습니다. 학생 대화는 AI가 카테고리와 감정으로만 분류해서 집계를 만듭니다."
>
> "심지어 '같은 고민을 한 학생이 5명 미만이면' 아예 대시보드에 표시 안 합니다. 수련회에 5명만 참석했을 때 누구 한 명이 특정돼버리면 안 되니까요."

## 시각 제안
- 위→아래 흐르는 다이어그램 (3단계 파이프라인)
- 각 단계마다 자물쇠 🔒 아이콘
- "교사 불가" 빨간 X 표시

> "교사에게는 '자해 징후 1건'이라는 **카테고리만** 전달됩니다. 누가 썼는지, 뭐라고 썼는지는 여전히 비공개예요. 대신 교사가 상담 가이드를 바로 볼 수 있고, 긴급 표현은 관리자에게 별도 알림이 갑니다."

   ✉️ 수련회 후
   종합 리포트 → 심방 방향 논의
   30일 뒤 원문 자동 파기
```
